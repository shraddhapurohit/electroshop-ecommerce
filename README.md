# ElectroShop E-Commerce Website

An e-commerce website project built using PHP, MySQL, Bootstrap, HTML, CSS, and JavaScript.

## Features
- User Registration & Login
- Product Listing
- Cart System
- Checkout Page
- Admin Dashboard
- Product Management
- Order Management

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript

## How to Run
1. Install XAMPP
2. Place project in `htdocs`
3. Import database in phpMyAdmin
4. Start Apache & MySQL
5. Open `localhost/electroshop`
